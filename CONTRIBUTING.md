# Contributing

We welcome your input on this project.

## Bug Reporting

Please [open an issue](https://github.com/Shopify/shopify-app-php/issues) describing the issue with detailed reproduction steps.

## Feature Request

Please [open an issue](https://github.com/Shopify/shopify-app-php/issues) describing the feature you wish to see.

## Developing the packages

For now this package is not accepting open source contributions.
